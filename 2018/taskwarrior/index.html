Tutorial of Taskwarrior

发表于 2018-03-13 | 0 comments

Taskwarrior 是一款命令行的TODO应用，功能强大，且使用逻辑简单方便，命令行的操作非常适合程序员。
（基本上能在命令行解决的问题就不要用GUI， 能用键盘解决的就不要用鼠标。）

目前我使用到的功能有：

  • 添加task
  • 给task修改所属的project、截至日期
  • 更改task的状态，如：start、done
  • 修改task的priority
  • 添加tags给task
  • 给task添加task依赖关系
  • …

基本使用

  • 添加 task

  • 修改task状态

  • 修改task信息

    其中due常用的选项有：

    • eom : end of month

  • 添加tags

其中有个特殊的tag是：next， 这回提升这个task的优先级。

  • 添加依赖关系

命令格式

task [filter] [command] [modifications | miscellaneous]

如果将Task比作一个类，可以把每个task都理解成一个实例。我们将实例存储在一个文件中，task就是用来处理这个文件的改动、信息筛选的命令。

  • filter-筛选器
    通过一些条件，筛选出我们每次操作的对象。

如果有多个条件，可以使and、or来进行连接
